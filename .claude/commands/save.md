# Save Conversation to Memory

この会話で重要な情報をSerenaのメモリに保存してください。

以下の手順で行ってください：
1. この会話で議論した重要な内容を要約する
2. `mcp__serena__write_memory` ツールを使って、適切なファイル名でメモリに保存する
3. ファイル名は内容を反映した分かりやすい名前にする（例: `conversation_2024-01-15_feature_discussion.md`）

保存する内容には以下を含めてください：
- 議論したトピック
- 決定事項
- 実装した変更点
- 今後のタスクやTODO
- その他重要な情報
